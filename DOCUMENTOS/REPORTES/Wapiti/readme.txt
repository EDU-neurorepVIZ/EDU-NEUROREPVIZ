1. Instalar wapiti3 usando pip3 install wapiti3
2. Correr el comando wapiti -u http://imagine-neurorepviz.eastus.cloudapp.azure.com:3000/
3. Correr el comando wapiti -u http://imagine-neurorepviz.eastus.cloudapp.azure.com:3000/session
4. Se genera un reporte en alguna carpeta. 
5. Abrir el archivos html iimagine-neurorepviz.eastus.cloudapp.azure.com_3000_12232020_2136.html en la máquina local para ver la prueba del punto 2.
6. Abrir el archivo html imagine-neurorepviz.eastus.cloudapp.azure.com_3000_12232020_2138.html en la máquina local para ver la prueba del punto 3.
